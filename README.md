# 🏋️ Fitness & Health Tracking Analysis Dashboard

## 📌 Project Overview
This project analyzes fitness and health tracking data to uncover meaningful insights about user activity, including steps, calories burned, sleep patterns, hydration, and heart rate.  

The workflow includes data cleaning using Python, exploratory data analysis (EDA), and building an interactive dashboard in Microsoft Power BI.

---

## 🎯 Objectives
- Clean and preprocess raw fitness data  
- Perform EDA to identify key patterns  
- Build an interactive dashboard for insights  
- Enable user segmentation (Age, Gender, Workout Type)  

---

## 🧹 Data Cleaning (Python)
- Handled missing values using mean/median imputation  
- Standardized categorical values (e.g., Gender: F → Female)  
- Converted data types (Date, numeric fields)  
- Created new features:
  - Age Group  
  - Sleep Category  
  - Water Category  
  - Workout Hour  

---

## 📊 Key Insights
- 🔥 Cardio workouts burn the most calories  
- ⏰ Peak activity occurs around 10 AM  
- 💤 Sleep duration shows minimal impact on activity levels  
- 💧 Hydration levels have slight influence on steps  
- 📈 User activity trends fluctuate across days  
- 🏋️ Workout types are fairly distributed among users  

---

## 📈 Dashboard Features
### KPIs
